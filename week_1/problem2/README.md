# 문제 2

## 문제설명

1월1일이 금요일일 때 입력값 a, b에서 a는 월, b는 일이고, 윤달이 있을 때 요일을 출력

## 풀이

1월 1일부터 a월 b일이 며칠 간격인지 계산하고 7로 나눈 나머지 만큼 요일을 더하기

### 코드 

```
def solution(a, b):
    answer = ''
    week=['FRI','SAT','SUN','MON','TUE','WED','THU']
    days=[31,29,31,30,31,30,31,31,30,31,30,31]
    day_sum=sum(days[:a-1])+b-1
    answer=week[day_sum%7]
    return answer
```
